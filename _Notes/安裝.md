# 安裝

01. forge安裝

    ```powershell
    npx create-electron-app@latest my-new-app --template=webpack-typescript
    ```

02. 新增（.yarnrc.yml）

    ```yaml
    nodeLinker: node-modules
    ```

03. 升級yarn

    刪除package.lock（將改為yarn.lock）

    ```powershell
    yarn set version stable
    yarn install
    ```
